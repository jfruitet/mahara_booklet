Nouvelle compétence

Pour chaque compétence saisir les valeurs :

  • Domaine: Une désignation identique pour toutes les compétences d'un même domaine.
  • Code : C'est une clé pour désigner une compétences dans un domaine donné.
  • Type: Permet de moduler l'affichage 0: Entête (aucun barème n'est appliqué), 1: Item standard ; 2: Item masqué, qui ne sera pas affiché du tout.
  • Description: Liste des savoir-faire et des habiletés que cette compétence démontre.
  • Barème : Une liste de valeurs séparées par une virgule, par exemple "Non acquis, Notion, Application, Maîtrise, Expertise";
  • Seuil Position du barèmes qui permet de valider la compétence ; par exemple pour le barème ci-dessus il faut indiquer '3' qui correspond à "Application".
    Si le barème n'est pas saisi il sera repris du barème par défaut "Non acquis, Notion, Application, Maîtrise, Expertise".
