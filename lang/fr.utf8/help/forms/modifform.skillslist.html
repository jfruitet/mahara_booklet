Liste de compétences

Pour chaque compétence saisir une ligne de valeurs respectant le motif suivant : 

code1;description1;[valeur_bareme11,valeur_bareme12,valeur_bareme13,...,valeur_bareme1N|seuil1]
code2;description2;[valeur_bareme21,valeur_bareme22,valeur_bareme23,...,valeur_bareme2N|seuil2]
...
ATTENTION:
code doit être une clé pour tout le serveur ; si le barème n'est pas renseigné c'est la valeur par défaut [Non acquis,Notion,Application,Maîtrise,Expertise|3] qui est assignée à la compétence.
