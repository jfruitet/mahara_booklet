Liste de compétences

Pour chaque compétence saisir une ligne de valeurs respectant le motif suivant : 

code1;type1;description1;[valeur_bareme11,valeur_bareme12,valeur_bareme13,...,valeur_bareme1N|seuil1]
code2;type2;description2;[valeur_bareme21,valeur_bareme22,valeur_bareme23,...,valeur_bareme2N|seuil2]
...
ATTENTION:
  • Domaine: Une désignation identique pour toutes les compétences d'un même domaine.
  • Code : doit être une clé pour tout le serveur ;
  • Type: 0: Entête (pas de barème appliqué), 1: Item; 2: masqué.
  • Barème : [valeur_bareme11,valeur_bareme12,valeur_bareme13,...,valeur_bareme1N|seuil1]
    Si le barème n'est pas renseigné c'est la valeur par défaut [Non acquis,Notion,Application,Maîtrise,Expertise|3] qui est assignée à la compétence.
  • Seuil Position du barèmes qui permet de valider la compétence ; par exemple pour le barème ci-dessus il faut indiquer '3' qui correspond à "Application".
