List of Skills

For each skill please type in a line of separated values matching the following pattern:
[optional scale]
domain; type1; code1; description1; [scale_value11, scale_value12, scale_value13, ..., scale_value1N|threshold1]
domain; type 2; code2; description2; [scale_value21, scale_value22, scale_value23, ..., scale_value2N|threshold2]
...

NOTA BENE:

  • domain: The same string for all skills for a peculiar domain of skills.
  • type: 0: Header (no scale), 1: Item; 2: Hidden.
  • code has to be a key for a peculiar domain;
  • scale a list of assessment values;
  • threshold Index of the 'passed' value for the used scale.
  • If the list of scale values is "Needs more work, Meets the mark, Going beyond, Super!" then the threshold is 2 which match the "Meets the mark" position.
  • If the scale values are not set they are kept from the so called "Default Skill Assessment Scale".
