New skill

  • domain: The same string for all skills for a peculiar domain of skills.
  • type: 0: Header (no scale), 1: Item; 2: Hidden.
  • code has to be a key for a peculiar domain.
  • description: the know-hows and the expertises linked to this skill.
  • scale a list of assessment values;
  • threshold Index of the 'passed' value for the used scale.
  • If the list of scale values is "Not relevant, Needs more work, Meets the mark, Going beyond, Super!" then the threshold is 3 which match the "Meets the mark" position.
    Attention: 'Not relevant' is mandatory in first position to allow that a skill will not be selected at all...
  • If the scale values are not set they are kept from the so called "Default Skill Assessment Scale".
