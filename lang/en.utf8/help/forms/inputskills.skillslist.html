List of Skills

For each skill please type in a line of separated values matching the following pattern:
[optional scale]
domain; code1; description1; [scale_value11, scale_value12, scale_value13, ..., scale_value1N|threshold1]
domain; code2; description2; [scale_value21, scale_value22, scale_value23, ..., scale_value2N|threshold2]
...

NOTA BENE:
domain: The same string for all skills for a peculiar domain of skills.
code has to be a key for a peculiar domain;
If the scale values are not set they are kept from the so called "Default Skill Assessment Scale".
